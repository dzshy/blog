## 安装

```
# yum install cronie
# systemctl start crond
# systemctl enable crond
```

## 编辑定时执行表

```
$ crontab -e
```

## 每周日和周一一次

```
0 0 * * 0-1
```

## 每周二和周五一次

```
0 0 * * 2,5
```

## 每周日一次

```
0 0 * * SUN
```

## 每两个月一次

```
0 0 1 */2 *
```

## 每年八月十五日一次

```
0 0 15 8 *
```

## 每天3点一次

```
0 3 * * *
```

## 每两小时一次

```
0 */2 * * *
```

## 每小时一次

```
0 * * * *
```

## 每十五分钟一次

```
0/15 * * * *
```

## 每分钟一次

```
* * * * *
```
